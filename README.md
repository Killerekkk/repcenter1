## 🚀 Hosting on GitHub Pages

1. Push this repo to GitHub (see setup below)
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live at `https://<username>.github.io/<repo-name>`

---

## 🛠️ Quick Setup

```bash
# 1. Clone or init
git init
git add .
git commit -m "Initial RepCenter site"

# 2. Create repo on GitHub and push
gh repo create repcenter --public --push --source=.
# OR manually:
git remote add origin https://github.com/<username>/repcenter.git
git branch -M main
git push -u origin main
```
